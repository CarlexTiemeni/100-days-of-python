# Day 2 — Tip Calculator

This project calculates how much each person should pay when splitting a bill including a tip.

### How the Program Works

The program asks the user for:

- The total bill amount
- The tip percentage (10%, 12%, or 15%)
- The number of people splitting the bill

The program then:

1. Converts the tip percentage into a decimal  
2. Calculates the total tip amount  
3. Adds the tip to the original bill  
4. Splits the total between the selected number of people  

### Example

```
Total bill: $150
Tip: 12%
People splitting the bill: 5

Each person should pay: $33.60
```

### What I Learned

While building this project I practiced:

- Converting data types using `int()` and `float()`
- Performing mathematical calculations
- Formatting numbers with `round()`
- Writing Python logic step by step
